# Singer Soundboard App (Flutter)

A fun Flutter app that plays a song when you tap on a singer's image. Built with the `audioplayers` package.

## Features
- Play audio on image tap
- Clean and modern UI
- Responsive layout for web and mobile

## Getting Started

1. Clone this repo
2. Run:
   ```bash
   flutter pub get
   flutter run
